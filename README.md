# scals

### Usage

```
python scals.py [ GR 1 test_sets/c.txt [ EQ 3 test_sets/a.txt test_sets/a.txt test_sets/b.txt ] ]
{2, 3}
python scals.py [ LE 2 test_sets/a.txt [ GR 1 test_sets/b.txt test_sets/c.txt ] ]
{1, 4}
```
